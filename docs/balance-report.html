③ 튜닝안별 7일차 도달 스테이지

핵심 발견

  1. 지수적 벽: Day 1에 30층 폭발 후 하루 +1씩. 적 성장(1.14ⁿ) > 보상 성장(1.12ⁿ) + 강화비용 폭증이 겹침.
  2. 곡선 상수(적/보상) 조정만으로는 벽이 안 풀림 → 진짜 원인은 지출 곡선과 곱셈형 루프 부재.
  3. 환생이 장식용이었다: state.prestige가 포인트만 쌓고 보너스 미적용 → 수정(수입 배수)하니 곡선이 개선(37→50층).
  4. 더 깊은 구조적 병목: 유닛 파워가 레벨 상한(랭크×20)에 묶여, 상한 개방은 소환(희소 자원)에 의존 → 수입만으로는 못 넘음.
