② 수정 — 환생 파워 배수로 벽 해소

환생 OFF는 ~36층에서 정체(평평). 환생 ON(기본)은 벽 없이 꾸준히 등반, 종합 튜닝안은 더 깊이 도달.

③ 튜닝안별 7일차 도달 스테이지

레버를 겹칠수록 도달 깊이가 커짐. 환생 루프가 단일 최대 레버.

진단 → 수정 → 재검증

  1. 문제(벽): 파워가 레벨 상한(랭크×20)에 묶이고, 유일한 돌파 수단인 환생이 수입만 늘리고 파워는 안 늘려 지수적 난이도(1.14ⁿ)를 못 따라감 → 최저달성률 44%로 뒤처짐, 병목 5회.
  2. 수정: 환생에 상한 없는 글로벌 파워 배수 추가(prestigePowerBonus). 벽에서 환생 → 배수로 더 깊이 재등반하는 정통 방치형 루프. 진행도(peakStage)는 유지.
  3. 결과: 벽 해소. 36층 → 57층, 최저달성률 44% → 112%, 병목 5회 → 0회(급정체 없음).
  4. 모든 상수는 core/balance.mjs 단일 소스 → 시뮬레이터가 값을 바꿔가며 곡선을 계속 튜닝 가능.
  5. 재점검(유물·펫 추가 후): 계정 성장(환생×유물×펫)이 accountMods로 합산돼 7일 만에 계정 배수 ×30까지 누적. 곡선은 여전히 병목 0회로 매끄럽고, 계정성장 OFF 대비 +21층 더 깊이 도달. 지수 난이도 탓에 배수 폭증 대비 stage는 완만히 상승(방치형 특유의 숫자 인플레이션 — 정상).
